Frequently Asked Questions

What is ClawBox Lite exactly?
ClawBox Lite is the everyday, accessible-first version of ClawBox — private AI hardware by OpenClaw Hardware. It runs on a NVIDIA Jetson Orin Nano 8GB with 67 TOPS of AI performance, 512GB NVMe storage, and comes with OpenClaw pre-installed. At €549 one-time, it brings professional local AI to everyone without requiring a technical background or monthly subscription fees.
How is ClawBox different from running AI on my laptop?
Your laptop's CPU or integrated GPU is not designed for AI inference. It will be slow (often 1–3 tokens per second for larger models), run hot, drain battery, and tie up your machine for other tasks. ClawBox uses NVIDIA's dedicated AI chip (NPU) with 67 TOPS of purpose-built neural processing — it runs AI at 15 tokens per second efficiently at just 15W, 24/7, without affecting any other device in your home. It also stays on while your laptop is closed, asleep, or away.
Does ClawBox require a subscription or cloud account?
No. ClawBox is a one-time purchase at €549. There are no mandatory subscriptions, no required cloud account, and no recurring fees. Optionally, you can connect ClawBox to cloud APIs (OpenAI, Anthropic, Gemini) to use their models for certain tasks — but this is entirely optional. The default experience is 100% local and fully offline-capable.
What models does ClawBox run locally?
ClawBox runs optimized versions of popular open-weight models via OpenClaw and Ollama. This includes Llama 3, Mistral, Phi-3, Gemma, and other models optimized for the Jetson Orin Nano architecture. The 8GB RAM supports models up to 7B parameters natively, with quantized versions of larger models also supported. OpenClaw manages model selection automatically based on the task at hand.
Can I install my own software on ClawBox?
Yes. ClawBox runs standard Linux (Ubuntu) on top of NVIDIA's Jetson platform. You have full root access if you want it. You can install additional software, Python packages, Docker containers, or any tool you need. The device is yours — it is not locked down or restricted. OpenClaw runs as a system service but does not prevent you from using the hardware however you like.
Is 8GB of RAM enough for modern AI?
For local AI assistant tasks — chat, document analysis, email, browsing automation, smart home control — 8GB is more than sufficient. The Jetson Orin Nano's unified memory architecture and NPU let it punch above its weight class compared to general-purpose hardware with the same RAM. For extremely large models (70B+), you would need cloud APIs; ClawBox can optionally route those requests to cloud providers while handling privacy-sensitive tasks locally.
What happens if OpenClaw releases a new version?
Updates are free. OpenClaw is open-source software, and ClawBox customers receive all updates at no additional cost. Updates can be applied from the web dashboard in one click, or automatically if you enable auto-updates. Your configuration, skills, and data are preserved across updates.
